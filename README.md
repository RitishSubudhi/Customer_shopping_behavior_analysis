# Customer_shopping_behavior_analysis
An end-to-end data analysis project focused on understanding customer shopping behaviour using real transaction data. The project includes data cleaning, SQL-based analysis, customer segmentation, and interactive dashboard creation using PowerBI to support data-driven business decisions.
